# Brand Kit – CargoClick

Referencia rápida para aplicar en Canva y cualquier herramienta de diseño.

---

## Colores

| Nombre | HEX | Uso |
|--------|-----|-----|
| Azul primario | `#0A2A5E` | Fondos oscuros, headers, títulos principales |
| Azul secundario | `#0B3D91` | Botones, acentos, gradientes |
| Verde acento | `#1F7A5C` | CTAs, elementos destacados |
| Fondo claro | `#F5F7FA` | Fondos de sección, slides de contenido |
| Texto principal | `#1A1A1A` | Cuerpo de texto sobre fondo claro |
| Texto suave | `#6B7280` | Descripciones, subtítulos |
| Blanco | `#FFFFFF` | Texto sobre fondos azules |

## Gradiente de marca

- Dirección: 135°
- Inicio: `#0A2A5E`
- Fin: `#0B3D91`

---

## Tipografía (Canva)

| Uso | Fuente | Peso |
|-----|--------|------|
| Títulos grandes | Plus Jakarta Sans | ExtraBold 800 |
| Subtítulos | Plus Jakarta Sans | Bold 700 |
| Cuerpo | DM Sans | Regular 400 |
| Etiquetas / badges | DM Sans | SemiBold 600 |

## Prompt base para Canva Magic Design

Pega esto al inicio de cualquier prompt para mantener coherencia de marca:

> "Estilo corporativo moderno, profesional pero cercano. Colores azul oscuro #0A2A5E y azul #0B3D91 para portada y cierre, fondo blanco o gris claro #F5F7FA para slides de contenido. Tipografía bold para títulos, sans-serif limpia para cuerpo. Empresa: CargoClick, transporte de carga terrestre en Colombia, audiencia B2B."
